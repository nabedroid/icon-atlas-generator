# 🎨 Icon Atlas Generator: 画像たちのシェアハウスへようこそ！

散らかった画像ファイルたちに頭を抱えていませんか？「ああっ！アイコンがあちこちに！」なんて叫びたくなる夜もあるでしょう。
そんなあなたに朗報です！この **Icon Atlas Generator** は、散らばった画像たちをギュギュッと一枚の画像（アトラス）にまとめ上げる、デジタル界の収納アドバイザーです！✨

## 🚀 なにこれ？

これはただのアトラスジェネレーターではありません。画像たちの「テトリス」を勝手にやってくれる凄腕マネージャーです。

*   **アトラス生成**: 複数の画像をアップロードすると、彼らが仲良く一枚の画像に収まります。「お前、もっと詰めろよ！」なんて喧嘩は起きません。
*   **自動トリミング**: 透明な余白？そんな無駄なスペースは許しません。ダイエット成功後のようにスッキリさせます。
*   **パディング設定**: 画像同士が近すぎて気まずくならないように、適切なソーシャルディスタンス（パディング）を設定できます。
*   **プレビュー**: 出来上がりをリアルタイムで確認。「思ってたんと違う...」という悲劇を防ぎます。

## 🛠️ 使っている魔法（技術スタック）

このツールは、以下の魔法で動いています。

*   **React**: UIを作るための魔法の杖。
*   **TypeScript**: 型安全という名のプロテクター。バグというモンスターから守ってくれます。
*   **Vite**: 光の速さでビルドする韋駄天。
*   **MUI (Material UI)**: 見た目を良くするオシャレ服。
*   **Docker**: どこでも同じ環境を再現する「どこでもドア」。

## 🎮 遊び方

### 🏠 ローカルで遊ぶ (Node.js愛好家向け)

Node.jsという相棒がいるなら、こう唱えてください：

```bash
# 呪文の準備（依存関係のインストール）
npm install

# 召喚！（開発サーバー起動）
npm run dev
```

そして、魔法の鏡（ブラウザ）で `http://localhost:5173` を覗いてみてください。そこには新しい世界が広がっています。

### 🐳 鯨の上で遊ぶ (Docker愛好家向け)

コンテナの海を泳ぐのが好きなら、こちら：

```bash
# クジラを呼ぶ
docker-compose up --build
```

`http://localhost:3000` にアクセスすれば、コンテナの中でパーティーが始まっています。
